# Инструкция по установке и использованию

## Установка
1. Клонируйт�� репозиторий:
   ```bash
   git clone https://github.com/ftint091-cmd/polymarket-agent.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd polymarket-agent
   ```
3. Установите необходимые зависимости:
   ```bash
   npm install
   ```

## Использование
После установки вы можете запустить агент следующей командой:
```bash
node index.js
```  

Настройте необходимые параметры в конфигурационном файле перед запуском.
